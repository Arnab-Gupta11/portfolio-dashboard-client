## Installation

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Steps to Set Up the Project Locally

#### **1. Clone the Repository**

```bash
https://github.com/Arnab-Gupta11/Portfolio-client.git
cd Portfolio-client
```

#### **2. Install Dependencies**

```bash
npm install
```

#### **3. Start the development server**

```bash
npm run dev
```
